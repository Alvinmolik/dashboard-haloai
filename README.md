# Dashboard Aqiqah Nurul Hayat x Halo.AI

Dashboard monitoring leads & konversi dari data Halo.AI.

## Deploy ke Vercel

Ikuti tutorial lengkap yang disertakan bersama project ini.

## Update Data

Untuk update data baru, minta file `App.jsx` baru dari Claude dengan menyertakan CSV terbaru.
Kemudian replace file `src/App.jsx` dan push ke GitHub — Vercel otomatis rebuild.

## Tech Stack
- React 18 + Vite
- Recharts (charts)
- Deploy: Vercel (free tier)
